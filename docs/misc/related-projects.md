# Related projects

This page contains a non-exhaustive list of the current ld-decode related projects as well as forks and add-ons.

If you would like to add further links to this page please feel free to [contribute via a PR](contributing.md).

## ld-decode
[ld-decode](https://github.com/happycube/ld-decode){target="_blank"} - LaserDisc decoder

[decode-orc](https://github.com/simoninns/decode-orc){target="_blank"} - a cross-platform orchestration and processing framework for LaserDisc and tape decoding workflows

[encode-orc](https://github.com/simoninns/encode-orc){target="_blank"} - a PAL and NTSC encoder capable of producing ld-decode and vhs-decode compatible TBC files (and SQLite metadata) to assist with testing

[ld-decode-tools](https://github.com/simoninns/ld-decode-tools){target="_blank"} - The legacy decoding tool chain (replaced by decode-orc)

[The Domesday Duplicator](https://github.com/simoninns/DomesdayDuplicator){target="_blank"} - the ld-decode reference LaserDisc RF sampling hardware
